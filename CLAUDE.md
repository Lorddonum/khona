i am a developper, i want to create a website for a lighting company, the company name is "KHONA" and the logo is the image "KHONA.jpg". The website is an e-commerce website that will be in four languages: arabic, french and english. The website will have the following pages : 
1. Home : this page will contain a slider of the products, a list of the products, and a contact form. 
2. Products : this page will contain all the products of the company, the products are divided into categories, each category has a name and a description, each product has a name, a description, a price, a quantity, a category, and a image. 
3. Contact : this page will contain a contact form.
4. About : this page will contain information about the company. 
5. Admin : this page is private, accessible only through a secure link, it will contain a dashboard to manage the products, the categories, the orders, the customers, and the users. 

The stack will be : 
- Frontend : React 
- Backend : Node.js 
- Database : MongoDB 
- Authentication : JWT
- Payment : Stripe

The design will be modern, clean, and user-friendly. The color palette will be a little dark with a dark brown as primary color and white/beige as secondary color. I want the website to be responsive on all devices.

The project will be developed using best practices, code will be clean, well-organized, and well-documented.
